# Notebooks

This folder is reserved for Jupyter notebooks used during exploratory
data analysis and model experimentation.

## Suggested Notebooks

| Notebook | Purpose |
|---|---|
| `01_eda.ipynb` | Exploratory data analysis — distributions, correlations, missing values |
| `02_feature_engineering.ipynb` | Visualising engineered features and their impact |
| `03_model_comparison.ipynb` | Side-by-side model result visualisations |

## Running Notebooks

```bash
pip install jupyter
jupyter notebook
```

## Note

The core pipeline (`src/`) is designed to run as standalone Python scripts
without notebooks. Notebooks here are for exploration and visualisation only.
